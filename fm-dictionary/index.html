Core Features

Everything you need
to master vocabulary

Roadmap Learning

Structured learning path with chapters, stages, and progressive difficulty. Track your journey from beginner to advanced.

Pronunciation Practice

Real-time voice comparison powered by on-device audio analysis. No audio stored — instant feedback, total privacy.

Gamification System

8 unlockable badges, daily streaks, and quiz scoring keep learners motivated. All achievements sync across devices.

Study Groups

Join private study groups to share learning progress with peers. Progress is visible within the group only — not public.

Personal Dictionary

Save and bookmark words as you learn. Your personal word collection syncs with Hive for instant offline access.

Multilingual UI

Full interface localization in Vietnamese and English via easy_localization — including badges, labels, and all system text.

App Permissions

What the app accesses
and exactly why

Microphone

Used exclusively for the pronunciation practice feature. Audio is processed on-device in real-time for comparison with reference audio. No recording is stored, transmitted, or retained after the session ends.

Required for feature

Internet Access

Syncs learning progress to Firebase, enables Google / Apple sign-in, and fetches content updates. The core learning experience works fully offline once content is downloaded.

Required for sync

Push Notifications

Sends optional daily learning reminders and streak alerts to keep you on track. Can be disabled at any time in your device Settings without affecting any App functionality.

Optional
